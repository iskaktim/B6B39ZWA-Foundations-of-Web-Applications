# Assignment of the Semester Project

## **Assignment**

**Title:** Discussion Forum

**Goal:** Provide a platform where a community of users can create posts, share content, and interact through comments.

## **User Roles**

1. **Guests**
2. **Registered Users**
   * Regular Users
   * Administrator
   * Owner 

## **Description**

### **Core Functionality**

* A post contains a title, content, author information, creation date, optional image, and last-posted timestamp.
* Users can comment on posts; each comment includes the author, text, and timestamp.
* The homepage displays the newest posts, sorted by time, with pagination.
* Each post has its own detail page containing full content and a comment section.

## **Role-Based Capabilities**

### **Guests**

* Can browse all posts and comments.
* Can register or log in.
* Cannot create posts or comments.

### **Regular Users**

* Can create, edit, and delete their own posts.
* Can add comments and manage their own comments.
* Can edit their profile information and avatar.

### **Administrator**

* Can delete any post or comment.
* Can promote users to administrators.
* Can access the Admin Panel.

### **Owner**

* Full administrative control.
* Can delete any user.
* Can assign or revoke admin roles.
