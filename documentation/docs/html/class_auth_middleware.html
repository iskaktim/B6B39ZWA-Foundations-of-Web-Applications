Detailed Description

AuthMiddleware

Provides access control checks for different user roles. Ensures that only authenticated users, admins, or owners can perform certain actions or access restricted pages.

AuthMiddleware::requireOwnerOrAdmin ( int $owner_id)
static

Require the user to be the owner, an admin, or the owner of the resource.

This is used for actions such as deleting posts or comments. Users with the "owner" or "admin" role are always allowed. Otherwise, only the resource owner (matching user_id) may proceed.
