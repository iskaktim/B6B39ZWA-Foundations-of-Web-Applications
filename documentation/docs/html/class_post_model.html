PostModel::getPostsPaginated ( int $page,
int $perPage,
?int $userId = null )

Retrieve paginated posts.

If $userId is provided, fetches only posts by that user. Otherwise, retrieves all posts in the system. Posts are ordered by most recent update.

Parameters
int$pageThe current page number.
int$perPageNumber of posts per page.
int | null$userIdOptional ID of a user to filter posts.
Returns
array Returns an array containing:
  • 'posts': A list of post records with usernames.
  • 'pagination': Metadata including current page, total pages, post count, and navigation flags.
